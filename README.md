## Installation

- Important: You must have installed the React.js and Git

```bash
# clone it
$ git clone git@github.com:airton-canale/chat-firebase.git

$ cd chat-firebase

# Install dependencies
$ npm install

# Running scripts
$ npm start
```
- And enter in your browser using http://localhost:3000
